CORS FIX:
  OLD: @CrossOrigin(origins = {"http://localhost:3000", "http://localhost:5173"})
  NEW: @CrossOrigin(origins = {"http://localhost:*"})
  WHY: Allows frontend on any localhost port, not just specific ones

================================================================================
                        TROUBLESHOOTING GUIDE
================================================================================

ISSUE: "Cannot connect to backend server"
  CAUSE: Backend not running or wrong port
  SOLUTION: Check if Spring Boot is running on 8081
  COMMAND: curl http://localhost:8081/api/auth/test

ISSUE: "No GPF records found"
  CAUSE: Database has no data
  SOLUTION: Run INSERT_TEST_GPF_DATA.sql
  COMMAND: sqlplus aditya/aditya2468@xe @INSERT_TEST_GPF_DATA.sql

ISSUE: "CORS error" in browser console
  CAUSE: CORS not configured properly
  SOLUTION: Rebuild backend with fixes
  COMMAND: mvn clean install && mvn spring-boot:run

ISSUE: "Network request failed"
  CAUSE: Frontend URL doesn't match backend
  SOLUTION: Check .env file for correct API_BASE_URL
  FILE: drdo-FrontEnd/.env

ISSUE: "Database connection failed"
  CAUSE: Oracle not running
  SOLUTION: Start Oracle database service
  COMMAND: sqlplus aditya/aditya2468@xe
